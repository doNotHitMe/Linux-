# Linux常用命令--基本文件管理（上）

文件/目录是我们最经常打交道的对象，也是存放、管理数据重要工具。最基本的文件/目录无外乎增、删、查、改，接下来介绍常用的命令。

## 1.创建

`mkdir [-p|-m] dirName`

- p：若要创建的目录的上级目录不存在，则一并创建上级目录；

- m：设置目录的权限；

- dirName：目录名，多个则用空格隔开

- 例子：`mkdir -p -m 777 /home/abc`

  

`touch fileName`

- fileName：文件名，多个则用空格隔开
- 例子：`touch a.txt b.txt`
